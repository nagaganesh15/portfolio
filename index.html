Memory Card Game Click here

Overview

The Memory Card Game is a simple and interactive web-based game designed to test and improve the player's memory skills. The objective is to find all matching pairs of cards in the least number of steps and shortest time possible.

Features

  • Interactive UI: The game interface is user-friendly with visually appealing buttons and backgrounds.
  • Dynamic Gameplay: The positions of the cards are randomized each time the game starts, ensuring a unique experience every time.
  • Timer: The game includes a timer that tracks the time taken to complete the game.
  • Step Counter: The game keeps track of the number of steps taken by the player to complete the game.
  • Restart Functionality: Players can restart the game at any time to try and improve their performance.

Technologies Used

  • HTML: Provides the structure of the game interface.
  • CSS: Styles the game elements, including buttons, background, and layout.
  • JavaScript: Implements the game logic, including randomizing card positions, handling user interactions, and tracking game progress.

How to Play

  1. Start the Game: Click the "Play" button to start the game. The timer will begin, and the cards will be ready for interaction.
  2. Flip the Cards: Click on any card to flip it and reveal the image underneath. Try to remember the image and its position.
  3. Find Matches: Click on another card to find its matching pair. If the cards match, they will remain visible. If they do not match, they will be flipped back over.
  4. Track Your Progress: The number of steps taken is displayed on the screen. Aim to complete the game in the fewest steps possible.
  5. Restart if Needed:If you want to try again, click the "Restart" button to reset the game.

Conclusion

The Memory Card Game is a fun and engaging project that showcases the use of HTML, CSS, and JavaScript to create an interactive web application. It is an excellent example of how to implement game logic and user interaction in a web-based environment. This project not only provides entertainment but also helps in improving memory skills.

Catch the Water Drops GameClick here

Overview

"Catch the Water Drops" is an engaging browser-based game where players must catch falling water drops while avoiding ice cubes. The game challenges players to move a glass left and right to catch the water drops and increase their score. With vibrant visuals, responsive controls, and an escalating difficulty level, the game promises an enjoyable experience for players of all ages.

Features

  • Responsive Design: The game is designed to work seamlessly on both desktop and mobile devices, ensuring a smooth and enjoyable user experience.
  • Interactive Gameplay: Players use arrow keys or touch controls to move the glass left and right. The goal is to catch falling water drops in the glass to score points while avoiding ice cubes.
  • Scoring System: Each water drop caught increases the player's score. The highest score achieved is saved and displayed, encouraging players to beat their previous best.
  • Game Mechanics: Water drops and ice cubes fall from the top of the screen at varying speeds. Catching an ice cube will break the glass. Players start with three glasses, and the game ends after all glasses are broken.
  • Audio and Visual Effects: Engaging sound effects and background music enhance the gaming experience. Visual feedback is provided for catching water drops and breaking glasses.
  • Instructions and User Interface: Clear instructions are provided at the start of the game. The user interface includes score displays, highest score display, and a restart option after the game ends.

How to Play

  1. Click the "Start" button to begin the game. Instructions on how to play are displayed before starting.
  2. Controls: On desktop: Use the left and right arrow keys to move the glass. On mobile: Move the glass by dragging it left and right on the screen.
  3. Objective: Catch as many water drops as possible to increase your score. Avoid catching ice cubes to prevent breaking the glass.
  4. Scoring: Your current score and highest score are displayed on the screen. The highest score is saved locally, allowing you to try and beat it in subsequent games.
  5. Game Over: The game ends when all three glasses are broken. Players can restart the game by clicking the "Start Again" button.

Technical Highlights

  • HTML & CSS: Provides the structure and styling for a visually appealing and responsive design.
  • JavaScript: Implements game logic, including water drop and ice cube generation, collision detection, and scoring.
  • Local Storage: Used to save and retrieve the highest score, providing a persistent high score feature.

Conclusion

Developing "Catch the Water Drops" was a fantastic learning experience, enhancing my skills in web development and game design. The project is designed to be both fun and challenging, offering a rewarding experience for players. I hope you enjoy playing the game and look forward to your feedback.
